## Compiling and Running

Execute the `run.sh` file to compile and run the program.
```bash
$ bash run.sh
```
Alternatively, execute the `makefile` to compile,
```bash
$ make -j$(nproc)
```
and run the program with:
```bash
$ ./program.out
```
